Layout

Das Layout einer Webseite beschäftigt sich unter anderem damit, wie einzelne Bilder, Texte, usw. angeordnet sind. Mithilfe von CSS kannst du das Layout deiner Webseite nahezu komplett bestimmen. Leider ist dies nicht immer so einfach und nachvollziehbar.

Was früher sehr aufwendig war, geht mittlerweile doch etwas leichter von der Hand mithilfe des Flexbox Layouts. Dies ermöglicht es dir, flexibler Sachen zu platzieren und kann mithilfe von display: flex; aktiviert werden. Ist ein Element nun flex, kannst du die Elemente, welche sich direkt in diesem Befinden, die sogenannten Kinder, mit einfachen Befehlen herumschieben. Das Element, welches flex ist, nennt man häufig auch Container, da es die Kinder umschließt und diese sich innerhalb des Containers bewegen können. Als Analogie kannst du dir einen Schiffcontainer vorstellen, wo du festlegen kannst wie du die Ware darin stapelst.

Ist der Container nun flex, kannst du beispielweise mit flex: 10% bestimmen, wie viel Platz jedes der Kinder innerhalb des Containers einnehmen soll - hier 10%. Für folgendes HTML 

<div id="container">
  <p class="kind">Kind 1</p>
  <p class="kind">Kind 2</p>
  <p class="kind">Kind 3</p>
  <p class="kind">Kind 4</p>
  <p class="kind">Kind 5</p>
</div>
kannst du beispielsweise mit 
div#container {
  display: flex;
}
p.kind {
  flex: 20%;
}
festlegen, dass jedes der fünf Kinder 10%, also ein fünftel, des verfügbaren Platzes für sich beanspruchen kann. Besonders praktisch ist auch diese Kombination. 
<div id="container">
  <div id="mittig">Dieser Text ist dann mittig ausgerichtet</div>
</div>

div#container {
  display: flex;
}
div#mittig {
  margin: auto;
}
Es zentriert ein Element, hier <div>, mittig im flex-container.
Hinweis: <div> sind besondere Elemente. Sie bringen für sich gesehen keinen Mehrwert zu HTML, da man sie auf der Webseite nicht sieht und sie keine eigene Funktion haben. Jedoch bringen sie Struktur in deine Webseite. Du kannst eines oder mehrere Elemente, wie beispielsweise einen Block Text, damit gruppieren und mittels CSS den Style des gesamten Blocks anpassen. Es gibt auch noch <span>, welches die gleiche Wirkung wie <div> hat. Während <div> meist für größere Blöcke verwendet wird, verwendet man hingegen <span> eher für einzelne Wörter.

Deine Aufgabe:

  1. In der <section> zur Zubereitung findest du bereits eine korrekte Gruppierung der Elemente mittels <div>. Mache das äußerste <div> in dieser Section zu einem flex-Container. Die beiden inneren <div> sollen jeweils 50% des Platzes einnehmen.
  2. Das <div>, welches das Bild umschließt, soll auch zu einem flex-Container werden. Zentriere mit dessen Hilfe das Bild des Pizzaofens.
