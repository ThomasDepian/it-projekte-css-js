JS: Schleifen

Wir wollen nun jene Zutaten auslesen, welche angeklickt sind. Dazu verwenden wir eine Schleife. Schleifen können dazu verwendet werden, gewissen Code mehrmals auszuführen. Ein typisches Beispiel wäre hierfür das Ausführen eines Codes für jedes Element einer Liste. Es gibt in JavaScript viele verschiedene Arten Schleifen zu schreiben, aber jene, welche wir verwenden, sind folgendermaßen aufgebaut. 

let summe = 0;
for (const nummer of [1,5,10,0]) {
  summe = summe + nummer;
}
Die Variable nummer wird im Laufe der Zeit jedes Element der Liste [1,5,10,0] genau einmal repräsentieren. Man spricht hierbei auch von "über die Liste iterieren". In diesem Beispiel wird nur die Summe der Elemente gebildet; es kann jedoch jeder beliebige Code innerhalb der Schleife sein.

Du hast bereits Variablen kennengelernt. Variablen können von verschiedenen Datentypen sein, je nachdem was sie repräsentieren sollen. Beispiele hierfür sind Zahlen, oder Zeichenketten, sogenannte Strings. Wir werden in dieser und den folgenden Aufgaben viel mit Strings arbeiten. Diese speichern Zeichenketten, also einfach Text und können so erstellt werden (achte auf die ""): 

let string1 = "";
let string2 = "hallo";
let string3 = " welt";
let string4 = string2 + string3;
Wie du sehen kannst, kann man auch zwei Strings mittels des + aneinander reihen, so wie wenn du zwei Zahlen addieren würdest.

In der vorherigen Aufgabe hast bereits eine Art gelernt, wie man HTML-Elemente von JavaScript aus ansprechen kann. Schauen wir uns eine weitere Möglichkeit an. Du hast im CSS-Teil schon einige Selektoren geschrieben, welche gewisse Elemente auswählen und ihr Aussehen verändern. Dies wäre beispielsweise ein Selektor, der alle <h2>-Überschriften mit der Klasse untertitel auswählt. 

h2.untertitel {}
Mit der Funktion document.querySelectorAll() kannst du solche CSS Selektoren auch in JavaScript verwenden. Dieser Befehl wählt alle Elemente aus, welche in dieses Muster fallen. Dies können, im Gegensatz zu document.getElementById(), auch mehrere sein. Deshalb wird dir auch eine Liste mit Elementen zurückgegeben, über welche du dann mit einer Schleife iterieren kannst. Der Code für den oben angegebenen CSS-Selektor würde so ausschauen. 
let elemente = document.querySelectorAll('h2.untertitel');;
for (const element of elemente) {
  // code
}

Deine Aufgabe:

  1. Erstelle eine Textvariable zutaten.
  2. Selektiere von der Webseite alle Checkboxen (<input type="checkbox">) die ausgewählt (markiert) wurden. Dies kannst du mittels input[type=checkbox]:checked machen. Speicher dir diese Elemente in der Variable zutatenBoxen.
  3. Für jedes dieser Elemente in zutatenBoxen, hänge deren Namen an den String zutaten dran. Auf den Namen der Zutat zutat kannst du mittels zutat.name zugreifen. Zwischen den Strings kannst du jeweils die Zeichenkette ", " dazuhängen, damit es leserlich bleibt.
  4. Bisher wurde ja beim Klick auf den Knopf Speichern der Titel der Pizza ausgegeben. Nun soll "Pizza [NAME DER PIZZA]; Zutaten: [INHALT DER VARIABLE zutaten]." ausgegeben werden.
