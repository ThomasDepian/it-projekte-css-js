JS: If-Abfragen

Zu guter Letzt wollen wir uns noch mit if-Abfragen befassen. Diese haben folgende Struktur. 

if (es ist heute sonning) {
  ich gehe heute an den Strand
} else {
  ich gehe heute ins Schwimmbad
}
Der Code innerhalb der runden Klammern (es ist heute sonning) wird als Bedingung bezeichnet. Ist diese Bedingung war, wir der Code im ersten Block ausgeführt, ansonsten jener im zweiten Block. Das bedeutet, falls es heute sonnig ist, gehe ich an den Strand, sonst in das Schwimmbad. Diese if-Abfragen erlauben uns auf unterschiedliche Ereignisse, beispielsweise der Eingabe einer Besucherin oder eines Besuchers deiner Webseite, jeweils angemessen zu reagieren.
Tipp: Falls du im "sonst"-Fall keinen Code ausführen möchtest, kannst du den Block auch komplett weglassen: 
if (es ist heute sonning) {
  ich gehe heute an den Strand
}

Wir haben bereits viele Selektoren kennengelernt. Dies ist nun der dritte, und letzte, Selektor den wir uns gemeinsam anschauen. Mit dem Selektor document.getElementsByName() kannst du Elemente anhand des Namens auswählen. Jedoch handelt es sich hier nicht um den Namen des Tags, bspw. <div> oder <p>, sondern um den Namen des Eingabefeldes. Wie du in der HTML Datei sehen kannst, hat (fast) jedes Eingabefeld einen Namen (Attribut name; <input name="hallo">). Dies ist wichtig, damit du später die Eingabe auch zum Feld zuordnen kannst. Wir können aber auch in JavaScript Elemente anhand des Namens, z.B. geschlecht, selektieren: 

let elemente = document.getElementsByName("geschlecht");

Deine Aufgabe:

  1. Wähle die Radiobuttons anhand des name-Attributs aus. Das name-Attribut hat den Wert käse.
  2. Finde heraus, welches der beiden Radiobuttons ausgewählt ist. Dies kannst du entweder mit einer Schleife machen, oder indem du explizit den ersten bzw. zweiten Radiobutton auswählst. Mit dem Status .checked kannst du abfragen, ob ein Radiobutton ausgewählt ist. Je nachdem welcher Radiobutton ausgewählt ist, hänge bitte hinten an die Zutatenliste mit Käse oder ohne Käse an.
Tipp: Mit [0] und [1] kannst du auf das erste und zweite Element einer Liste zugreifen.
Tipp: Du kannst mit der Eigenschaft .value den Wert des ausgewählten Radiobuttons abfragen. Der ist bereits mit Käse oder ohne Käse.
Bonus: Falls kein Namen für die Pizza eingegeben wurde, so füge das erstellte <p>-Element nicht zum HTML Dokument hinzu, sondern gib eine Warnung aus. Die Anzahl der Zeichen in einem String kannst du mit .length abfragen und eine Warnung mit dem Text Achtung gibst du mit alert("Achtung") aus.
