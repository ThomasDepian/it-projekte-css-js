Zustandsänderungen der Elemente

Jedes der HTML-Elemente kann in verschiedenen Zuständen sein. Du kannst CSS-Regeln festlegen, welche nur dann angewendet werden, wenn das Element in dem von dir angegebenen Zustand ist. Vermutlich am häufigsten wird dabei der Zustand hover verwendet. Ein Element ist immer dann im Zustand hover, wenn sich die Maus gerade über ihm befindet. Eine CSS-Regel, welche nur in diesem Zustand greift, kannst du folgendermaßen festlegen. 

#hover-mich {
  color: red;
}
#hover-mich:hover {
  color: green;
}
Ist das Element nicht in diesem Zustand, greift die Regel nicht. Dies würde bedeuten, dass im obigen Beispiel die Schriftfarbe des Elements mit der ID hover-mich solange rot ist, bis du mit der Maus drüber fährst; dann wird es grün.

Deine Aufgabe:

  1. Der Link, welcher zum Kuppelofen führt, soll standardmäßig folgenden Stil aufweisen. 
    color: initial;
text-decoration: underline;
  2. Wenn man mit der Maus drüber fährt, soll er folgendermaßen ausschauen. 
    font-weight: 900;
Hinweis: Wir haben nicht alle der oben verwendeten Regeln kennengelernt. inital ist eine vordefinierte Anweisung, welche besagt, dass der ursprüngliche Wert für die Regel verwendet werden soll. Dies bedeutet in diesem Fall konkret, dass die blaue Schrift, welche standardmäßig vom Browser erzeugt wird, ignoriert werden soll, und der Link, wie der restliche Text, schwarz dargestellt wird. Mit text-decoration kann das Erscheinungsbild des Textes weiter verändert werden. underline legt hier beispielsweise fest, dass der Text unterstrichen werden soll. font-weight: 900 kommt dir vielleicht bekannt vor. Mit dem kannst du nachträglich den Stil (oder besser gesagt die dicke) der Schriftart ändern. Wir haben dies auf der Webseite von Google Fonts bereits gesehen.
