Textlayout

CSS bietet dir einige Möglichkeiten an, die Art, wie der Text in Erscheinung tritt, zu verändern. Dazu gehören unter anderem die Ausrichtung des Textes und die Schriftart.

Die Textausrichtung kannst du mittels TODO ändern. Normalerweise ist der Text linksbündig ausgerichtet, wie dieser hier. Dies kannst du mittels TODO machen. Willst du den Text rechtsbündig ausrichten, so verwende TODO. Offizielle Texte werden häufig im sog. Blocksatz geschrieben. Dabei werden die Abstände und Zeilenumbrüche zwischen den Wörtern so angepasst, dass der Text sowohl links- als auch rechtsbündig ist. Dies geschieht mittels TODO. Überschriften hingegen sind oft in der Mitte der Seite bzw. des Blockes, zentriert. Um alle h2-Überschriften zu zentrieren, so schreibe beispielsweise folgendes. TODO

Aktuell verwendet der Browser eine Standardschriftart. Jedoch kannst du die Schriftart selber bestimmen.
Google Fonts beispielsweise bietet eine Vielzahl an Schriftarten an, welche du in deiner Webseite einbinden kannst. Hierfür einfach auf der Übersichtsseite eine Schriftart auswählen, beispielsweise LAto, auswählen. Wenn du nach unten Scrolls siehst du eine Vorschau der Schriftart, mit all den angeboteten Stilen. "thin" oder "light" sind dünnere Schriftdicken (oder auch Stile genannt) dieser Schriftart, "regular" ist der Standard und "bold" und "black" sind die dickeren Stile.
An der Zahl kannst du auch erkennen, wie dick die Schrift ist. Für den normalen Text wird eine Schriftdicke von 400 oder 500 verwendet. "Italic" sind kursive Schriftstile. Mit einem Klick auf das plus wählst du den Stil aus. TODO FOTO. Es wird automatisch ein Code erzeugt, den du auf deine Webseite kopieren musst, damit du die Schriftart verwenden kannst. Falls du keine Übsersicht siehst, so klicke hier. TODO FOTO. Den Code musst du dann kopieren, und in den head-Tag hineinkopieren, z.B.: TODO FOTO TODO CODE Willst du dann auf ein Element, beispielsweise einen Absatz, eine andere Schriftart anwenden, so musst du nur den Code, welcher dir weiter unten von Google angezeigt wird, bei der von dir gewünschten CSS-Regel angeben. Beispielsweise so. Soll deine gesamte Webseite in der gleichen Schriftart sein, so kannst du sie auch gleich auf den body anwenden. Wie du vermutlich gesehen hast, ist alles, was du im Browser sehen kannst, innerhalb des body.
