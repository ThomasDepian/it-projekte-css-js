Textlayout

CSS bietet dir einige Möglichkeiten an, die Art, wie der Text in Erscheinung tritt, zu verändern. Dazu gehören unter anderem die Ausrichtung des Textes und die Schriftart.

Die Textausrichtung kannst du mittels <h1> ändern.

Normalerweise ist der Text linksbündig ausgerichtet, wie dieser hier. Dies kannst du mittels text-align: left; machen.

Willst du den Text rechtsbündig ausrichten, so verwende text-align: right;.

Offizielle Texte werden häufig im sogenannten Blocksatz geschrieben. Dabei werden die Abstände und Zeilenumbrüche zwischen den Wörtern so angepasst, dass der Text sowohl links- als auch rechtsbündig ist. Dies geschieht mittels text-align: justify;.

Überschriften hingegen sind oft in der Mitte der Seite bzw. des Blockes, zentriert.
Um alle <h2>-Überschriften zu zentrieren, so schreibe beispielsweise folgendes. 

h2 {
  text-align: center;
}

Du kannst auch die Schriftart selber bestimmen.
Google Fonts beispielsweise bietet eine Vielzahl an Schriftarten an, welche du in deiner Webseite einbinden kannst. Hierfür einfach auf der Übersichtsseite eine Schriftart auswählen, beispielsweise Lato, auswählen. Wenn du nach unten Scrolls siehst du eine Vorschau der Schriftart, mit all den angeboten Stilen. thin oder light sind dünnere Schriftdicken (oder auch Stile genannt) dieser Schriftart, regular ist der Standard und bold oder black sind die dickeren Stile. An der Zahl kannst du auch erkennen, wie dick die Schrift ist. Für den normalen Text wird eine Schriftdicke von 400 oder 500 verwendet. italic sind kursive Schriftstile. Mit einem Klick auf das plus wählst du den Stil aus.

Es wird automatisch ein Code erzeugt, den du auf deine Webseite kopieren musst, damit du die Schriftart verwenden kannst. Falls du keine Übersicht siehst, so klicke hier (oben rechts auf der Seite). Den Code musst du dann kopieren, und in den <head>-Tag hineinkopieren, z.B.: 
<head>
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Lato&display=swap" rel="stylesheet">
</head>
Willst du dann auf ein Element, beispielsweise einen Absatz, eine andere Schriftart anwenden, so musst du nur den Code, welcher dir weiter unten von Google angezeigt wird, bei der von dir gewünschten CSS-Regel angeben. Beispielsweise so. 
p {
  font-family: 'Lato', sans-serif;
}
Soll deine gesamte Webseite in der gleichen Schriftart sein, so kannst du sie auch gleich auf den <body> anwenden. Wie du vermutlich gesehen hast, ist alles, was du im Browser sehen kannst, innerhalb des <body>.

Deine Aufgabe:

  1. Ändere die Schriftart auf Roboto. Du kannst aber selbstverständlich auch eine andere Schriftart auswählen.
  2. Des Weiteren sollen die Überschriften <h1> und <h2> zentriert sein.
  3. Der Absatz zur Zubereitung einer Pizza (ohne dem Bild), soll im Blocksatz (justify;) dargestellt werden.
