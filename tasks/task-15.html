JS: HTML Manipulation

Ein Vorteil von JavaScript ist, dass man auf Elemente im HTML zugreifen kann, deren Text auslesen, oder wie du später sehen wirst, sie auch manipulieren kann.

Einerseits kann man Elemente im HTML Dokument auswählen. Dafür brauchen wir sogenannten Selektoren, welche wir in dieser und den kommenden Aufgaben kennenlernen werden. Ein Selektor wählt Elemente von der HTML-Seite anhand von gewissen Mustern aus. 
let element = document.getElementById('titel');
Dieser Selektor wählt beispielsweise das Element aus, welches die ID titel hat. Das ausgewählte Element wird in der Variable element gespeichert - der Variablenname kann aber von dir frei gewählt werden. Er darf aber nicht Sonderzeichen (!?äö etc.) oder Leerzeichen beinhalten.

Haben wir Elemente ausgewählt, können wir deren Status abfragen oder auch manipulieren. Um auf die Eigenschaften eines, in einer Variable gespeicherten, (HTML) Elementes zuzugreifen, benutzen wir die "."-Notation. Sei element die Variable von oben, welches ein HTML Element beinhaltet, und eigenschaft eine Eigenschaft des Elementes, so können wir so die Eigenschaft abfragen. 

let wertVonEigenschaft = element.eigenschaft;
Und so können wir der Eigenschaft einen neuen Wert zuweisen: 
element.eigenschaft = neuerWert;

Wir können aber nicht nur bereits bestehende Elemente auswählen, sondern auch neue Elemente erzeugen. Mit dem Befehl document.createElement("p") erzeugen wir ein neues <p>-Element, sprich einen Absatz. Dieses Element beinhaltet aber noch keinen Text. Um diesen zu setzen, überschreiben wir die Eigenschaft innerText: 

let neuesElement = document.createElement("p");
  neuesElement.innerText = "Hallo du!";
Das Element existiert bis jetzt nur in der Variable, der Browser kennt es so noch nicht. Was wir aber machen können ist, das von uns erstellte Element an ein anderes Element dranhängen. Dies machen wir mit der Funktion appendChild(); z.B. so: 
let neuesElement = document.createElement("p");
  neuesElement.innerText = "Hallo du!";
  let element = document.getElementById('titel');
  element.appendChild(neuesElement);
So wir das neue <p>-Element an das bereits bestehende Element hinten dran gehängt. Wie du siehst haben wir eine Funktion aufgerufen, so wie du sie in der vorherigen Aufgabe erstellt hast.
Achtung: Wenn du ein Element irgendwo dranhängst wird es zwar vom Browser erkannt, die HTML-Datei wird aber nicht verändert. Das bedeutet auch, dass beim Neuladen das erstellte Element nicht mehr vorhanden ist.

Deine Aufgabe:

  1. Gib dem Eingabefeld für den Namen der Pizza eine ID, damit du es von JavaScript aus selektieren kannst.
  2. Erstelle im HTMl-Dokument unterhalb der Eingabeform (nach </form>) ein <div>-Element mit einer ID. Hier werden wir bei Knopfdruck was erzeugen.
  3. Immer wenn auf den Knopf (<button>) <Speichern> gedrückt wird, soll folgendes passieren.
    1. Wähle (Selektiere) das Eingabefeld aus und lese den Status value aus. Dieser beinhaltet den aktuell in diesem Feld eingegebenen Text.
    2. Erstelle ein <p>-Element und setze den Text (Status innerHTML) auf den Text im Eingabefeld.
    3. Dieses <p>-Element soll im von dir zuvor erstellten <div>-Element hinten drangehängt werden. Dazu musst du jedoch vorher das <div>-Element auswählen (selektieren).
