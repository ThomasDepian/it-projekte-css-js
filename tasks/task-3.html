Margin und Padding

Erinnerst du dich noch an das Box-Model, welches wir dir am Beginn dieser Einheit gezeigt haben? Jedes HTML-Element ist von einer Box umgeben. Die Abstände außerhalb und innerhalb dieser Box lassen sich dann mit dem Padding und Margin angeben.

Willst du beispielsweise, dass das Element mit der Klasse kein-margin keinen Margin aufweist, so kannst du dies so machen: 

.kein-margin {
  margin: 0;
}
Du kannst aber auch den Margin in jede Richtung individuell angeben. 
#titel {
  margin-left: 16px;
  margin-right: 20;
  margin-top: 0;
  margin-bottom: 0;
}
Somit hat beispielsweise das Element mit der ID titel oben einen Margin von 16px, rechts einen Margin von 20px und sonst keinen. Den Padding kannst du genauso steuern. Schreibe einfach padding anstatt margin.

Deine Aufgabe:

  1. Der <body> deines Dokumentes soll auf keiner Seite einen Margin haben.
  2. Auch für <h2>-Überschriften soll der Margin nach oben entfernt werden.
  3. Jede <section> soll folgenden Padding aufweisen:
    • Oben: 1em
    • Rechts und Links: 3em
    • Unten: 3em
  4. Nachdem wir den Margin für Überschriften entfernt haben, wollen wir einen Padding nach oben von 1em einfügen. Dadurch klebt die Überschrift nicht so am Rand der <section> (du wirst später sehen, was wir damit meinen).
  5. Der Absatz zur Zubereitung der Pizza soll in allen Richtungen einen padding von 10 Pixeln aufweisen.
