Margin und Padding

Errinerst du dich noch an das Box-Model, welches wir dir am Beginn dieser Einheit gezeigt haben?
Jedes HTML-Element ist von einer Box umgeben. Die Abstände außerhalb und innerhalb dieser Box lassen sich dann mit dem Padding und Margin angeben.

Willst du beispielsweise, dass das Element mit der Klasse kein-margin keinen Margin aufweist, so kannst du dies so machen: TODO CODE Du kannst aber auch den Margin in jede Richtung individuell angeben. TODO CODE Somit hat beispielsweise das Element mit der ID "titel" oben einen Margin von 16px, rechts einen Margin von 20px und sonst keinen.
Den Padding kannst du genauso steuern. Schreibe einfach padding anstatt margin.

Deine Aufgabe:

Der body deines Dokumentes soll auf keiner Seite einen Margin haben.
Auch für h2-Überschriften soll der Margin nach oben entfernt werden.

Jede Section soll folgenden Padding aufweißen:

  • Oben: 1em
  • Rechts und Links: 3em
  • Unten: 3em
Nachdem wir den Margin für Überschriften entfernt haben, wollen wir einen Padding nach oben von 1em einfügen.
Dadurch klebt die Überschrift nicht so am Rand der section (du wirst später sehen, was wir damit meinen).

Der Absatz zur Zuberitung der Pizza soll in allen Richtungen einen padding von 10 Pixeln aufweisen.
