Hintergrundbild

Als nächstes wollen wir ein Hintergrundbild zu einem Block hinzufügen. Die Syntax des Styles ist hier ähnlich wie bei der Hintergrundfarbe. Wir können dies so machen. 

div {
  background-image: url("https://bilder.com/hintergrund-bild.png");
}
Innerhalb von url geben wir den Pfad zum Bild an, ähnlich wie du es bereits bei Bildern auf der Webseite gemacht hast.

Der obige Code ist jedoch in den meisten Fällen nicht ausreichend, da er das Hintergrundbild nur "irgendwie" setzt. Um beispielsweise das Hintergrundbild mittig im Block zu platzieren, schreiben wir background-position: center;. Anstatt center können wir auch andere Werte, wie beispielsweise top einsetzen. Falls das Hintergrundbild zu klein ist, um den gesamten Block auszufüllen, so können wir festlegen, dass es öfters nebeneinander gereiht werden soll, um den gesamten Block zu füllen. Dies wird bei uns nicht notwendig sein, und wir können dies mit background-repeat: no-repeat; unterbinden. Dies hat aber den Nachteil, dass das Hintergrundbild unter Umständen nicht den gesamten Block ausfüllt. Wir können jedoch mit background-size: cover; festlegen, dass das Bild so vergrößert werden soll, damit es den ganzen Block ausfüllt.
Beachte: Um das Bild nicht zu verzerren, werden die Seitenverhältnisse des Bildes beibehalten. Dies kann zur Folge haben, dass nicht das gesamte Bild sichtbar ist.

Folgender Code würde beispielsweise für das Element mit der ID katzen dieses Hintergrundbild setzen. Dies bedeckt das gesamte Element, wird zentriert und nicht wiederholt. 

#katzen {
  background-image: url("https://upload.wikimedia.org/wikipedia/commons/5/59/Seal-tortie.jpg");
  background-position: center;
  background-repeat: no-repeat;
  background-size: cover;
}

Deine Aufgabe:

  1. Bitte gib der <section>, welche den Titel Pizza enthält, folgendes Hintergrundbild: https://cdn.pixabay.com/photo/2018/04/11/03/13/food-3309419_960_720.jpg.
  2. Es soll dabei zentriert sein und das gesamte Element bedecken. Für background-repeat, gebe bitte no-repeat ein.
  3. Damit das Bild gut zur Geltung kommt, setze bitte noch padding auf 0 in dieser Section.
Wenn du willst kannst du noch mit anderen Werten für background-position, background-repeat, oder auch background-size experimentieren.
