Hintergrundbild

Als nächstes wollen wir ein Hintergrundbild zu einem Block hinzufügen.
Die Syntax des Styles ist hier ähnlich wie bei der Hintergrundfarbe. Wir können dies so machen. TODO CODE Innerhalb von url geben wir den Pfad zum Bild an, ähnlich wie du es bereits bei Bildern auf der Webseite gemacht hast.

Der obrige Code ist jedoch in den meisten Fällen nicht ausreichend, da er das Hintergrundbild nur "irgendwie" setzt. Um beispielsweise das Hintergrundbild mittig im Block zu platzieren, schreiben wir TODO. Anstatt TODO können wir auch andere Werte, wie beispielsweise TODO einsetzen.
Falls das Hintergrundbild zu klein ist, um den gesamten Block auszufüllen, so können wir festlegen, dass es öfters nebeneinander gereiht werden soll, um den gesamten Block zu füllen. Dies wird bei uns nicht notwendig sein, und wir können dies mit TODO unterbinden. Dies hat aber den Nachteil, dass das Hintergrundbild nicht den gesamten Block ausfüllt. Wir können jedoch mit TODO festlegen, dass das Bild so vergrößert werden soll, damit es den ganzen Block ausfüllt.
Beachte: Um das Bild nicht zu verzerren, werden die Seitenverhältnisse des Bildes beibehalten. Dies kann zur Folge haben, dass nicht das gesamte Bild sichtbar ist.
Folgender Code würde beispielsweise für das Element mit der ID TODO das Hintergrundbild TODO setzen. Dies bedeckt das gesamte Element, wird zentriert und nicht wiederholt. TODO CODE.

Deine Aufgabe:

Bitte gib der Section, welche den Titel "PIZZA" enthält, folgendes Hintergrundbild. TODO. Es soll dabei zentriert sein und das gesamte Element bedecken. Für TODO, gebe bitte TODO ein. Damit das Bild gut zur geltung kommt, setze bitte noch TODO auf 0 in dieser Section.
