Schriftgrößen

Schriftgrößen kann man in verschiedenen Einheiten angeben.
Entweder absolut, beispielsweise in Pixeln (px), oder relativ zu der Schriftgröße von anderen Elementen (em oder rem). Wollen wir beispielsweise sagen, dass alle <h2>-Überschriften 16px groß sind, so können wir dies so machen: 

h2 {
  font-size: 16px;
}
Soll die Schriftgröße jedoch doppelt so groß sein wie bisher, schreiben wir: 
h2 {
  font-size: 2em;
}

Deine Aufgabe:

  1. Standardmäßig, das heißt im gesamten <body> deines HTML-Dokumentes, soll die Schriftgröße 16 Pixel betragen.
  2. Der Titel (<h1>), soll sieben mal so groß werden, alle Überschriften zweiten Grades (<h2>) doppelt so groß.
