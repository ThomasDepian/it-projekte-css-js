Bonus: Responsive Design

Wir verbringen immer mehr und mehr Zeit vor unseren Smartphones. Responsive Design wird daher immer wichtiger. Responsive Design hat sich zum Ziel gesetzt, dass sich das Layout der Webseite dem Gerät, auf welchem sie betrachtet wird, anpasst. Während beispielsweise zwei Spalten nebeneinander auf dem Laptop angezeigt werden, so sind diese auf dem Smartphone untereinander.

Eine Möglichkeit, wie man das erreichen kann, sind sogenannte Media Queries. Schauen wir uns zuerst ein Beispiel an. 

h2#mach-mich-rot {
  color: black;
}
@media (max-width: 500px) {
  h2#mach-mich-rot {
    color: red;
  }
}
Diese Media Query fungiert wie ein if (mehr zu if-Abfragen gleich im JS-Teil) in einer Programmiersprache. Falls die Bedingung oben erfüllt ist, hier, dass die Größe maximal 500 Pixel beträgt, so wird zusätzlich noch die angegebene Regel angewendet, hier beispielsweise eine Überschrift rot gefärbt. Die Regeln unten greifen beispielsweise nur, falls die Bildschirmbreite mindestens 600 Pixel beträgt. 
@media (min-width: 600px) {
  h2#mach-mich-rot {
    color: red;
  }
}

Wie du dir denken kannst, ist dies nicht sehr praktikabel. Deshalb gibt es seit kurzem das sogenannte Grid Layout. Es ist eine Alternative zu dem flex was wir zuvor gesehen haben und ermöglicht dir, Elemente in einem Grid anzuordnen. Dies kannst du dir wie eine Tabelle vorstellen und wird so verwendet. Mit der Regel display: grid; schalten wir das Grid-Layout auf dem Container-Element ein. gap: 10px; legt den Abstand zwischen den Zellen des Grids fest - hier 10px. Dies kannst du dir als den Abstand zwischen den Kind-Elementen vorstellen. Schließlich können wir mit grid-template-columns das Layout der Spalten festlegen. grid-template-columns: 100px 1fr 1fr; legt beispielsweise fest, dass es drei Spalten geben soll: Die erste ist 100 Pixel breit, und der verbleibende Platz wird auf zwei weitere Spalten aufgeteilt (1fr legt dies fest).

Deine Aufgabe:

  1. Falls die Bildschirmbreite unter 1200 Pixeln ist (also maximal 1200 Pixel beträgt), so soll es eine zusätzliche Regel geben, die die Höhe des Bildes mit dem Steinofen reguliert. Verwende hierfür height: auto;.
  2. Im Abschnitt zu den Fakten (Fakten zu Pizza) haben wir wieder einige <div>s vorbereiten. Das erste <div> soll nun ein Grid werden, wobei der Abstand zwischen den einzelnen Elementen 1em beträgt.
  3. Damit das Grid auch gut zur Geltung kommt, soll die Breite des Grids (width) 100% betragen.
  4. Für das Layout der Spalten setze: repeat(auto-fit, minmax(380px, 1fr)). Dies erzeugt so viele Spalten und Zeilen wie notwendig - abhängig von der Anzahl der Elemente und des verfügbaren Platzes. Jede Spalte besitzt somit auch mindestens eine Breite von 380 Pixeln.
  5. Füge noch justify-items: center; hinzu. Mit diesem Befehl werden die einzelne Elemente zentriert, was auf großen Bildschirmen besser zur Geltung kommt.
