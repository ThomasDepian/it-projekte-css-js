Bonus: Responsive Design

Wir verbringen immer mehr und mehr Zeit vor unseren Smartphones. Responsive Design wird daher immer wichtiger.
Responsive Design hat sich zum Ziel gesetzt, dass sich das Layout der Webseite dem Gerät, auf welchem sie betrachtet wird, anpasst.
Während beispielsweise zwei Spalten nebeneinander auf dem Laptop angezeigt werden, so sind diese auf dem Smartphone untereinander.

Wie du dir denken kannst, ist dies nicht sehr praktikabel. Deshalb gibt es seit kurzem das sogenannte Grid Layout. Es ist eine alternative zu dem flexbox was wir zuvor gesehen haben und ermöglicht dir, Elemente in einem Grid anzuordnen. Dies kannst du dir wie eine Tabelle vorstellen und wird so verwendent. Mit der Regel display grid schalten wir das Grid-Layout auf dem Container-Element ein. gap: 10px legt den Abstand zwischen den Zellen des Grids fest. Dies kannst du dir als den Abstand zwischen den Kindelementen vorstellen. Schließlich können wir grid-template-columns: 10px 1fr 1fr das Layout der Spalten festlegen. Dies legt beispielsweise fest, dass es drei Spalten geben soll: Die erste ist 100 Pixel breit, und der verbleibende Platz wird auf zwei weitere Spalten aufgeteilt (1fr legt dies fest).

Deine Aufgabe:

Falls die Bilschirmbreite unter 1200 Pixeln ist (also maximal 1200 Pixel beträgt), so soll es eine zusätzliche Regel geben, die die Höhe des Bildes mit dem Steinofen reguliert (Verwende hierfür height auto).
Im Abschnitt zu den Fakten (Fakten zu Pizza) haben wir wieder einige divs vorbereiten. Das erste div soll nun ein Grid werden, wobei der Abstand zwischen den einzelnen Elementen 1em beträgt. Damit das Grid auch gut zur geltung kommt, soll die Breite des Grids (width) 100% betragen. Für das Layout der Spalten setze: "repeat(auto-fit, minmax(380px, 1fr))". Dies erzeugt so viele Spalten und Zeilen wie notwendig - abhängig von der Anzahl der Elemente und des verfügbaren Platzes. Jede Spalte besizt somit auch mindestens eine Breite von 380 Pixeln. Füge noch justify-items: center; hinzu. Mit diesem Befehl werden die einzelne Elemente zentriert, was auf großen Bildschirmen besser zur geltung kommt.
