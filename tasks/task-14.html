JS Funktionen

Wir werden im folgenden beim Erzeugen einer neuen Pizza JavaScript Code von der HTML-Datei aus aufrufen. Dafür erzeugen wir eine Funktion, welche später die Logik beinhaltet. Das können wir folgendermaßen machen. 

function meineErsteFunktion() {
  // Code
}

Events sind eine Möglichkeit um JS-Funktionen von der HTMl-Datei aus aufrufen zu können. Ähnlich wie bei CSS, können auch hier gewisse Elemente auf spezielle Events reagieren, wie beispielsweise ein Button auf einen Mausklick. Mit folgendem Code können wir dem Button (<button>) sagen, dass beim Klick die Funktion hallo() aufgerufen werden soll. 

<button type="button" onclick="hallo()">Sag Hallo!</button>

Einige Elemente führen standardmäßig bspw. gewisse Operationen bei einem Klick durch. Unser Knopf von der HTML-Seite beispielsweise würde die Form an eine von uns angegebene Adresse abschicken. Wir wollen aber die Form mit JavaScript manipulieren. Um diese Standardfunktionalität auszuschalten, können wir die Art des Knopfes von type="submit" (= Abschicken) zu type="button" ändern, wie auch oben dargestellt.

Deine Aufgabe:

  1. Erzeuge in der Datei script.js eine Funktion neuePizza(). Diese Funktion soll beim Klick auf den Knopf (<button>) Speichern aufgerufen werden.
  2. Setze den Typ des Knopfes auf type="button", damit die Form nicht abgeschickt wird.
Tipp: Um zu testen, ob die Funktion funktioniert, kannst du folgenden Code hineinkopieren: alert("Funktioniert!");.
